# CCN-Lab

Experiments: Implement the following in C/C++
    1. For the given data, use CRC-CCITT polynomial to obtain CRC code. Verify the program for the cases
    i. Without error
    ii. With error.
    2. Write a program for a HDLC frame to perform Bit stuffing and destuffing a bit in a single frame
    3. Write a program for a HDLC frame to perform Byte stuffing and destuffing a bit in a single frame Character stuffing.
    4. Write a program for distance vector algorithm to find suitable path for transmission
    5. Implementation of Stop and Wait Protocol and Sliding Window Protocol
    6. Write a program for encryption and decryption of the text
